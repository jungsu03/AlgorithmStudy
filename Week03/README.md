학습 주제
DFS(Depth First Search) 탐색 알고리즘
BFS(Breadth First Search) 탐색 알고리즘
재귀 함수 및 완전탐색
그래프 탐색 실습 및 코드 리뷰

학습 내용
이번 주에는 그래프 탐색 알고리즘인 DFS와 BFS의 개념 및 동작 원리를 학습하였다. DFS는 재귀 함수와 스택 구조를 활용하여 깊이 우선으로 탐색하는 방법을 이해하였고, BFS는 큐(Queue)를 이용하여 가까운 노드부터 순차적으로 탐색하는 과정을 학습하였다. 또한 완전탐색의 개념을 익히고 다양한 예제 문제를 Python으로 구현하며 그래프 탐색 알고리즘의 활용 방법을 연습하였다. 이후 팀원들과 구현한 코드를 리뷰하며 다양한 풀이 방법과 개선점을 함께 확인하였다.

실습 코드
w3.py

어려웠던 점

DFS와 BFS의 탐색 순서를 구분하는 것이 처음에는 다소 헷갈렸으며, 특히 재귀 함수를 이용한 DFS 구현 과정에서 함수 호출 순서를 이해하는 데 시간이 필요하였다. 또한 그래프를 인접 리스트 형태로 표현하고 탐색하는 과정도 익숙하지 않아 여러 번 실습이 필요했다.

해결 방법

교재의 예제와 실습 문제를 반복적으로 구현하며 DFS와 BFS의 탐색 과정을 직접 확인하였다. 또한 팀원들과 서로의 코드를 비교하고 리뷰하면서 탐색 순서와 구현 방법의 차이를 이해할 수 있었고, 재귀 함수의 실행 과정을 단계별로 분석하여 알고리즘의 동작 원리를 익힐 수 있었다.

느낀 점

이번 학습을 통해 DFS와 BFS는 그래프 탐색뿐만 아니라 다양한 코딩테스트 문제에서 활용되는 핵심 알고리즘이라는 것을 알게 되었다. 특히 같은 문제라도 탐색 방법에 따라 접근 방식이 달라질 수 있다는 점이 인상적이었으며, 알고리즘의 원리를 이해하고 직접 구현하는 과정이 문제 해결 능력을 향상시키는 데 큰 도움이 되었다. 앞으로도 다양한 그래프 탐색 문제를 꾸준히 해결하며 알고리즘 활용 능력을 더욱 키워나가고자 한다.
